LABORATORIO 3.1 - Creación de una página web básica
====================================================
Curso: WADE 1000L - Front-End Technologies and UI
National University College

DESCRIPCIÓN:
Este laboratorio demuestra la creación de una página web básica utilizando
HTML5 y CSS. Se implementaron las siguientes características:

ARCHIVOS INCLUIDOS:
- index.html     : Página principal con estructura HTML5 completa
- styles.css     : Hoja de estilos externa
- README.txt     : Este archivo
- imagen.jpg     : Imagen de ejemplo

ESTRUCTURA HTML5 IMPLEMENTADA:
1. Sintaxis básica: etiquetas de apertura/cierre, atributos y valores
2. Etiquetas semánticas: header, nav, main, section, article, aside, footer
3. Características HTML5: canvas, figure, figcaption, formulario con validación
4. Comentarios en el código explicando cada sección

Sobre HTML5

HTML5 es la quinta versión del lenguaje de marcado HTML. Introduce nuevas etiquetas semánticas que mejoran la accesibilidad y el SEO de las páginas web.

Ventajas de HTML5

HTML5 ofrece mejor soporte para multimedia sin necesidad de plugins externos, mejora la semántica del código y añade nuevas APIs para aplicaciones web.
